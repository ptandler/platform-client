Interaktive Landkarte zur Bürgerbeteiligung

Diese interaktive Karte ermöglicht es, Fragen, Hinweise und Ideen mit Bezug zu einer Position auf der Landkarte zu sammeln. Dabei können die gesammelten Beiträge auch von anderen bewertet werden, als wie relevant sie diese selber einschätzen.

Erstellen eines Anliegens

Haben Sie Fragen, Hinweise oder Ideen, die bestimmte Bereiche in Ihrer Umgebung betreffen? Hier können Sie sie ganz einfach an uns weiterleiten.

Klicken Sie dafür auf die Position auf der Karte - oder auf den orangenen Plusbutton am linken Bildschirmrand. Wählen Sie anschließend aus den drei Kategorien die aus, die Ihr Anliegen beschreibt. Nun können Sie die Karte mit einem Klick an der Stelle markieren, auf die sich Ihr Anliegen bezieht. Mit dem Plus und Minus am linken Kartenrand können Sie in die Karte rein- oder rauszoomen. Fügen Sie Ihrem Anliegen im nächsten Schritt einen Titel und eine Beschreibung hinzu. Drücken Sie abschließend auf Absenden. Ihr Anliegen ist jetzt auf der Karte markiert.

Beiträge

Zu einer Übersicht über alle Beiträge gelangen Sie über den Button auf der linken Seite. Hier können Sie Beiträge mit dem „Daumen-hoch-Button“ priorisieren oder über das Drei-Punkte-Menü auf der rechten Seite die Beiträge zu Sammlungen hinzufügen. Detailliert einsehen können Sie einzelnen Beiträge durch einen Klick auf den Beitrag.

Sammlungen

Über den Button mit den sechs Quadraten links unten gelangen Sie zu einer Liste mit Sammlungen. Über die Suchfunktion oben können Sie nach bestimmten Sammlungen suchen. Z.B. Sammlungen, die eine bestimmte Stadt betreffen. Klicken Sie eine Sammlung aus der Liste an, um sich die entsprechenden Markierungen auf der Karte anzeigen zu lassen. Mit dem Button unten „Neu erstellen“ können Sie eigene Sammlungen hinzufügen.
